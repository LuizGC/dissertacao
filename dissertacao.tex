\resumo{As redes veiculares \emph{ad hoc} (\emph{Vehicular Ad Hoc Networks} - VANETs) são redes com alta mobilidade. Os veículos podem se comunicar um com os outros ou com infraestrutura na estrada. Comunicação sem fio permite novas aplicações que exploram a transferência de dados entre nós móveis. Redes oportunistas são redes onde a comunicação não é continua, isto é, nunca deve existir um caminho pré-definido entre o nó de origem e o nó destino. Isto proporciona suporte para os cenários em que os dados são transportados a partir de uma fonte para um destino através dos nós móveis que levam fisicamente os dados de um lugar para outro, com ou sem auxilio de uma infraestrutura fixa. Essas redes exigem a existência de um grande número de nós, de forma a aumentar a probabilidade de sucesso na entrega dos dados. Outra questão importante neste tipo de rede é a utilização de informações baseada no contexto que permitem uma decisão eficiente para transporte de dados entre os nós. O uso de agentes móveis representa uma tecnologia promissora para tratar os problemas aqui apresentados. O seu uso permite o encapsulamento de inteligência na rede, melhorando as decisões que serão responsáveis pelo trafego de dados. Essa abordagem oferece flexibilidade para elaborar estratégias de transporte dos dados, assim permitindo enfrentar possíveis problemas encontrados de forma mais ágil. A versatilidade do agente aumenta se os veículos possuirem sensores que possam captar a situação do condutor do veículo, por exemplo se ele esta embriagado avisar a policia. Outra aplicação é em caso de acidente o próprio veículo cria agentes que avisam os veículos vizinhos, cria outro agente para buscar um veículo da policia nas proximidades e outro para buscar uma ambulância mais próximos. O objetivo deste trabalho é analisar o impacto da infraestrutura no auxilio do agentes móvel a atingir seu objetivo. Outro objetivo e verificar o uso do ZigBee em redes veiculares para transportar agentes.}

\resumoingles{Vehicular Ad Hoc Networks - VANETs are networks with high mobility. The vehicles can communicate with each other or with the road infrastructure. Wireless communication enables new applications that exploit the data transfer between mobile nodes . Opportunistic networks are networks where communication is not continuous, that is, should never be a predefined path between the source node and the destination node. This provides support for scenarios where data is transported from a source to a destination via the mobile nodes to physically carry data from one place to another, with or without the aid of a fixed infrastructure. Such networks require the existence of a large number of nodes, so as to increase the probability of successful delivery of data. Another important issue in this type of network is the use of information based on context that enable effective decision to transport data between nodes. The use of mobile agents is a promising technology to address the problems presented here. It allows for the encapsulation of intelligence in the network, improving the decisions that will be responsible for data traffic. This approach offers flexibility to develop data transport strategies, thus enabling face possible problems encountered in a more agile way. The agent's versatility increases if the vehicles possess sensors that can capture the state of the driver of the vehicle, for example if he is drunk notify the police. Another application is in an accident the vehicle itself creates agents that warn neighboring vehicles, creates another agent to find a nearby police vehicle and another to seek a closer ambulance. The objective of this study is to analyze the impact of infrastructure on the help of mobile agents to achieve your goal. Another objective and check the ZigBee use in vehicular networks to transport agents.}
